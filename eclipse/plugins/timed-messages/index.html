Timed messages
Eclipse plugin

When we're developing, we constantly forget to do periodic things, like standing up, stretch, drink coffee, change keyboard in a pair programming session.

Timed messages eclipse plugin helps you to remember these tasks.

Get Started

How to install

Just search for Timed messages on the eclipse marketplace
or add the update-site www.marcelotomazini.com/eclipse/plugins/timed-messages/update-site

How to setup a reminder

Insert a new timer

Then choose Timed messages in any context menu

Choose a timer configuration

And a popup dialog will be fired to warn you
